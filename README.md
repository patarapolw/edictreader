# Edict reader

[EDICT2, JMdict](http://edrdg.org/jmdict/edict_doc.html) and [CEDICT](https://www.mdbg.net/chinese/dictionary?page=cedict) reader and searcher in dictionary format

## Installation

```
pip install edictreader
```
